Detailed Description

Main model for the address book view.

Contains fields and methods to enclose the functionalities of a ContactList, a TaggableList and a TrashCan. In particular, it allows for the storage and management of an ordered list of taggable SafeContacts ammitting omonyms, which can be restored within RecentlyDeleted#RETENTION_PERIOD_DAYS days of their deletion. It is created to grant full compatibility with a JavaFX UI
