Detailed Description

A class representing a contact in the address book.

A contact is defined by a name, a surname, a list of email addresses and a list of phone numbers. It can also be marked with an arbitrary number of tags, which are used to categorize contacts, and can show a profile picture. It is created to grant full compatibility with a JavaFX UI
