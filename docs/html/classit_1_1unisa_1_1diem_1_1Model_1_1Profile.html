Detailed Description

Represents a profile in the application.

A profile is characterized by a name, a phone number, a profile picture and an address book. Only the name field is mandatory. The name is subject to a maximum length of {@value MAX_NAMELEN} characters. The phone number is subject to the constraints of ItalianPhoneChecker. It is created to grant full compatibility with a JavaFX UI

name!=null
phone!=null
profilePicture!=null
addressBookPath!=null
